# Healthcare SQL Analysis Project

## Objective
Analyze patient visit and treatment data to derive healthcare insights using SQL.

## Tools Used
- SQL
- Joins
- Aggregations
- Window Functions

## Key Analysis Performed
- Identified patients with multiple visits
- Calculated average treatment cost per patient
- Determined most common diagnosis
- Analyzed monthly visit trends
- Ranked patients by total treatment expenditure

## Skills Demonstrated
- GROUP BY and HAVING
- Subqueries
- Window Functions (RANK)
- Healthcare Data Interpretation
- KPI and Cost Analysis
